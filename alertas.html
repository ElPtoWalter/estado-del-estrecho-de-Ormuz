Alertas y acceso a datos

Herramientas abiertas para seguir cambios sin depender de una única plataforma.

Alertas del navegador

Las alertas locales comparan el estado más reciente con el último visto en este dispositivo. Requieren permiso del navegador y no sustituyen un sistema profesional de avisos.

RSS

El feed RSS publica cambios y actualizaciones para lectores compatibles.

https://estrechoormuz.com/feed.xml

API JSON

El archivo de estado ofrece la clasificación, confianza, hora, evidencias y diagnósticos en formato legible por máquinas.

https://estrechoormuz.com/status.json

Limitaciones

La entrega de notificaciones depende del navegador, el sistema operativo y la conectividad. Para usos críticos crea tus propios controles sobre la API y consulta fuentes marítimas oficiales.
