Recibe avisos cuando cambie el estado

La web ofrece opciones sin registro y sin entregar tu correo electrónico.

Avisos en el navegador

El navegador comprobará status.json cada cinco minutos mientras esta pestaña permanezca abierta. Si cambia el estado público o la interpretación operativa, mostrará una notificación.

No es una notificación push permanente: al ser una web estática, la pestaña debe seguir abierta. Esta opción no instala software ni crea una cuenta.

Feed Atom / RSS

Suscríbete desde Feedly, Inoreader, Thunderbird, NetNewsWire o una automatización como IFTTT. Solo publica cambios significativos, no cada comprobación horaria.

https://elptowalter.github.io/estado-del-estrecho-de-Ormuz/feed.xml

API pública JSON

Otros proyectos pueden leer el estado, confianza, evidencias y última confirmación válida desde un archivo estático con CORS compatible con GitHub Pages.

https://elptowalter.github.io/estado-del-estrecho-de-Ormuz/status.json

Alertas del propietario

El workflow incluye integración opcional con Telegram y ntfy mediante secretos de GitHub. Cuando se configuran, solo se envía un mensaje si hay un cambio significativo.

IndexNow automático

Cuando cambia el estado, el workflow notifica automáticamente a los buscadores compatibles con IndexNow y les indica las páginas actualizadas. Las comprobaciones sin cambios no generan avisos innecesarios.

Indicador automatizado y conservador. No sustituye avisos oficiales de navegación, instrucciones de autoridades marítimas ni asesoramiento profesional.
