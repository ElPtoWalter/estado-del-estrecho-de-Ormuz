Monitoring

Get alerts when the status changes

The site provides options that require no account and do not ask for your email address.

Browser alerts

The browser checks status.json every five minutes while this tab remains open. If the public status or operational interpretation changes, it displays a notification.

This is not permanent push delivery: because the site is static, the tab must remain open. It does not install software or create an account.

Atom / RSS feed

Subscribe with Feedly, Inoreader, Thunderbird, NetNewsWire or an automation service such as IFTTT. It publishes meaningful changes only, not every hourly check.

https://elptowalter.github.io/estado-del-estrecho-de-Ormuz/feed.xml

Public JSON API

Other projects can read the status, confidence, evidence and last valid confirmation from a static file served by GitHub Pages.

https://elptowalter.github.io/estado-del-estrecho-de-Ormuz/status.json

Owner alerts

The workflow includes optional Telegram and ntfy integrations through GitHub secrets. Once configured, a message is sent only after a meaningful change.

Automatic IndexNow

When the status changes, the workflow automatically notifies search engines supporting IndexNow and lists the updated pages. Checks without changes do not create unnecessary submissions.

A conservative automated indicator. It does not replace official navigational warnings, maritime authority instructions or professional advice.
